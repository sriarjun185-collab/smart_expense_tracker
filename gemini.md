Project Name
Smart Expense Tracker
Tagline

"Track every rupee. Manage your money smarter."

Project Overview

Smart Expense Tracker is a web application that helps users manage their daily expenses and income. Users can register, log in, add transactions, categorize spending, view analytics, and monitor their monthly financial status through an interactive dashboard.

Technology Stack

Frontend

React + Vite
Tailwind CSS
React Router

Backend

Node.js
Express.js

Database

MongoDB Atlas (or local MongoDB)

Authentication

JWT
